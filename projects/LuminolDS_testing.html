Test Data¶

  • Input must be in a pandas dataframe format.
  • There must be at least two columns. One with the date keys and one with the values.
  • The dates must be either unparsed strings, or already pd.date_time objects.

Init Params¶

use_daynames_only = False, # Compute and return results only for days with the same name as most recent date
get_labels_only = False, # Whether to return labels or scores (labels:{0,1}, scores:{float})
get_lastday_only = False, # Return results for last day only
n_entries_threshold = 15, # Minimum number of datapoints to allow into detector
score_threshold=7, # Threshold to label anomalies based on Luminol's anomaly score
precision=4, # Luminol detector param
lag_window_size=1, # Days back to compare
future_window_size=1, # Days forward to compare
chunk_size=1, # Luminol detector param
